# cj360's keymap for the Bananasplit
## Layout Notes

    This switch layout is ANSI with the following modifications:
        Spilt Backspace
        Split right shift
        275 125 225 spacebar
        Bottom right singles


Base layer:
```
-------------------------------------------------------------------------------------------
| Esc |  1  |  2  |  3  |  4  |  5  |  6  |  7  |  8  |  9  |  0  |  -  |  =  |  \  | Del |
-------------------------------------------------------------------------------------------
|  Tab  |  Q  |  W  |  E  |  R  |  T  |  Y  |  U  |  I  |  O  |  P  | [ |  ]  | Backspace |
-------------------------------------------------------------------------------------------
|  Fn(Cpslock)  |  A  |  S  |  D  |  F  |  G  |  H  |  J  |  K  |  L  |  ;  | ' |  Enter  |
-------------------------------------------------------------------------------------------
|    Shift    |  Z  |  X  |  C  |  V  |  B  |  N  |  M  |  ,  |  .  |  /(shift) | Up | Fn |
-------------------------------------------------------------------------------------------
|  Ctrl  |  GUI |  Alt  |  Space  |  Fn(~)  |  Space   | Home | End | Left | Down | Right |
-------------------------------------------------------------------------------------------
```

Fn1 layer:

```
-------------------------------------------------------------------------------------------
|   | F1  |  F2  | F3  | F4  | F5  | F6  | F7  | F8  | F9  | F10 | F11 | F12 | PrtSc |    |
-------------------------------------------------------------------------------------------
|     |      | Pgup |     |    |     |     |     |     |   |     |     |     |      |     |
-------------------------------------------------------------------------------------------
|     | Home | Pgdn | End |    |     |     |     |     |     |Previous|Next|Play|         |
-------------------------------------------------------------------------------------------
|     |      |      |     |    |     |     |     |     |     |VOL-| VOL+ | Mute |         |
-------------------------------------------------------------------------------------------
|     |      |      |          |     |           |        |       |       |     |         |
-------------------------------------------------------------------------------------------
```

Build:
```
make thevankeyboards/bananasplit:cj360
```
